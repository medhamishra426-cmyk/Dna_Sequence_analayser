# DNA seqyence analyser

## Project Description
This project analyzes a DNA sequence using Python.

## Features
- Counts Adenine (A)
- Counts Thymine (T)
- Counts Guanine (G)
- Counts Cytosine (C)
- Calculates GC content
- Displays sequence length

## Technologies Used
- Python
- Google Colab
- GitHub
